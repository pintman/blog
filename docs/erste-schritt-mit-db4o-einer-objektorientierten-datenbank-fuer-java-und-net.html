Erste Schritte mit db4o - einer objektorientierten Datenbank für Java und .NET

Published: Mi 20 November 2013
By marco.bakera

In Sonstiges.

tags: Programmierung Video

db4o steht für "database for objects" und ist eine objekt orientierte Datenbank. Die Inhalte werden also nicht in Tabellen abgelegt, sondern direkt als Objekt gespeichert.

Ich habe mir das Paket angeschaut und ein Beispiel für Java programmiert und mitgefilmt. Das Ganze habe ich in der IDE Eclipse durchgeführt.
