25.11.-30.11.: Europe Code Week - denn: Programmieren macht Spaß

Date Mon 25 November 2013 Tags Bildung / Programmierung / Termin

Vom 25.11. bis zum 30.11. findet die europäische Code Week statt.

Coding is fun. Creative. Full of great people and passionate communities.

Organize a programming workshop, the one you planned to do in any case. Find a few friends who're curious about what you do and teach them how to code. Start a local study or support group for coding beginners. And remember, two is a group already.
