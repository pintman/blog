LTSP über Etherboot booten

Date 23.12.2013 Tags Informatik / Linux / Tools

In einem vorherigen Beitrag habe ich beschrieben, wie man einen LTSP-Server aufsetzt, der an PXE-fähige Clients ein Boot-Image ausliefert und dann startet.

Wenn die eingesetzte Netzwerkkarte nicht PXE-fähig ist, kann man auf das Projekt Etherboot/gPXE zurückgreifen. Es stellt bootfähige Images für CDs und Disketten zur Verfügung. Mit diesen Images kann der Prozess maßgeschneidert angepasst werden. So kann man z.B. auch über eine WLAN-Karte booten.
